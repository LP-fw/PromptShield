# PromptShield

Single Page Application sviluppata per il corso **Fondamenti di Sistemi Web** (2025/26).
Il progetto è stato realizzato impiegando:
- Vue.js 3
- Chart.js 4
- Bootstrap 5
- HTML5, CSS3, JavaScript

## Argomento scelto

Approfondimento della libreria **Chart.js** (libreria JavaScript open source per la creazione di grafici interattivi nel browser) applicata alla visualizzazione delle vulnerabilità in ambito AI.

## Viste

- **Cos'è Chart.js**: introduzione alla libreria, storia, caratteristiche e funzionamento
- **Approfondimento**: confronto con alternative, integrazione con Vue.js, configurazione avanzata e casi d'uso
- **Dashboard**: visualizzazione interattiva dei dati con 3 grafici Chart.js (severità, categorie, trend temporale)
- **Gestione Dati**: CRUD completo con persistenza tramite localStorage
